*A fast CLI for creating files and folders in your project without switching to your editor.*

TreeCraft helps you move quicker inside large or deeply nested codebases. Instead of manually navigating through directories to create files or folders, TreeCraft lets you:

- search for a folder anywhere in your project  
- pick the target directory  
- create files or folders instantly  
- avoid repetitive navigation steps  

Designed for developers who want a simple, fast, terminal-based workflow.
